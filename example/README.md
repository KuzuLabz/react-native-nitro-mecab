# Getting Started

Before building:

1. Download the [IPADIC dictionary](https://github.com/shogo82148/mecab/releases/latest) and extract.
2. Zip the contents into a file called `ipadic.zip`
3. Place `ipadic.zip` into the assets folder.

## Building

Run a prebuild first.
```bash
bunx expo prebuild
```

### Android
```bash
bun run android
```

### iOS
```bash
bun run ios
```

## Running
```bash
bun run start
```
